# 0005. 인스타그램 스타일 피드 전환

- **상태**: Accepted
- **날짜**: 2026-03-04

## 배경

Yuna의 피드는 사진 중심 콘텐츠(아기 사진, 가족 사진)를 다루지만,
초기 구현은 텍스트 위주 카드 레이아웃이었다.
사용자가 "피드에는 인스타그램을 최대한 참고하라"고 요청했다.

## 선택지

1. **현행 유지**: 텍스트 중심 카드에 작은 썸네일 첨부
2. **Pinterest 스타일**: 다양한 비율의 이미지를 Masonry 레이아웃으로 배치
3. **Instagram 스타일**: 1:1 정사각형 이미지 중심, 세로 스크롤 피드

## 결정

**Instagram 스타일**을 채택한다.

### 핵심 변경사항

| 영역 | 변경 전 | 변경 후 |
|------|---------|---------|
| 이미지 비율 | 자유 비율 (4:3) | **1:1 정사각형** (`aspect-square`) |
| 다중 이미지 | 미지원 | **최대 10장 캐러셀** (좌우 스와이프) |
| 게시물 작성 | 상단 인라인 버튼 | **하단 FAB(+)** → 전체화면 모달 |
| 피드 탐색 | 페이지네이션 | **무한 스크롤** |
| 좋아요 | 미지원 | **하트 토글** + 더블탭 |
| 댓글 | 별도 페이지 | **인라인 미리보기** (최근 2개) |

### 3단계 로드맵

- **Phase 1** (즉시): FE만으로 가능한 레이아웃 변경 — 1:1 비율, FAB, 무한스크롤
- **Phase 2** (현재 진행): BE+FE — 사진 업로드, 좋아요 API, 게시물 작성 모달
- **Phase 3** (Backlog): 이미지 최적화, 게시물 수정, 앨범 뷰

### 제약 조건

- 한 게시물 최대 **10장**
- 폐쇄형 원칙 유지: 이미지 직접 접근 불가, Presigned URL 필수
- 다운로드 버튼 미제공 원칙 유지

## 결과

- 피드가 사진 중심으로 전환되어 가족 SNS 성격에 부합
- FAB 버튼으로 게시물 작성 진입점이 항상 접근 가능
- BE에 Like 엔티티/API 추가 필요 → ADR-0006과 연계
