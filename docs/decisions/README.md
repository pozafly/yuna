# Architecture Decision Records (ADR)

프로젝트에서 내린 주요 결정을 기록합니다.

## 형식

각 ADR은 `NNNN-제목.md` 형태의 파일로 작성하며, 아래 구조를 따릅니다:

- **상태**: Proposed / Accepted / Deprecated / Superseded
- **배경**: 왜 이 결정이 필요했는가
- **선택지**: 어떤 대안들이 있었는가
- **결정**: 무엇을 선택했는가
- **결과**: 이 결정으로 인해 달라지는 것
