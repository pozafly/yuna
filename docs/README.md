## 핵심 원칙 요약 (절대 위반 금지)

1. **폐쇄형** — OWNER 초대 없이 접근 불가
2. **외부 공유 차단** — 인증·권한 없이 링크/이미지 직접 접근 불가
3. **서버 최종 권한 판단** — 프론트 숨김은 보안 아님
4. **다운로드 버튼 미제공** (단, Baby 삭제 후 30일 아카이브 다운로드 예외)
5. **원본 화질 유지, 워터마크 없음**
