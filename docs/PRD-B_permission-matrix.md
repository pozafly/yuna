# PRD-B: 권한 매트릭스

> **문서 상태**: 2차 업데이트 (2026-02-26)

범례: ✅ 가능 / ❌ 불가 / 🔒 조건부 (아래 주석 참조)

---

## 1. ROLE × 기능 매트릭스

### Baby 관리

| 기능 | OWNER | INVITED |
|------|:-----:|:-------:|
| Baby 생성 | ✅ | ❌ |
| Baby 정보 수정 | ✅ | ❌ |
| Baby 삭제 | 🔒¹ | ❌ |
| OWNER 초대 (공동 OWNER 추가) | ✅ | ❌ |
| INVITED 초대 발송 | ✅ | ❌ |
| 초대 재발송 | ✅ | ❌ |
| 초대 취소 | ✅ | ❌ |
| 구성원 차단 | ✅ | ❌ |
| 구성원 목록 열람 | ✅ | ❌ |
| 삭제된 Baby 데이터 다운로드 (30일 이내) | ✅ | ❌ |

> ¹ **Baby 삭제 정책 (확정)**: OWNER만 가능. UI에서 "삭제 후 복구 불가능" 명시 필수. 삭제 후 30일간 데이터 다운로드 가능, 30일 경과 후 완전 삭제.

| 콘텐츠 유형 | visibility | OWNER | INVITED (해당 Baby) | INVITED (미초대 Baby) | 차단된 사용자 | 탈퇴 사용자 |
|------------|-----------|:-----:|:-------------------:|:--------------------:|:------------:|:----------:|
| Post | PRIVATE | ✅ | ❌ | ❌ | ❌ | ❌ |
| Post | INVITED | ✅ | ✅ | ❌ | ❌ | ❌ |

## 3. 차단/탈퇴 후 기존 콘텐츠 표시 정책

| 상태 | 기존 댓글 UI | 기존 편지 UI | 접근 권한 |
|------|------------|------------|---------|
| 차단된 사용자 | "차단된 사용자의 댓글입니다." | "차단된 사용자의 편지입니다." | 열람·작성 모두 불가 |
| 탈퇴한 사용자 (INVITED) | (기존 콘텐츠는 유지, 당사자에게만 비노출) | (동일) | 완전 불가 |
| 탈퇴한 OWNER (공동 OWNER 중 1명) | (기존 콘텐츠 유지, 탈퇴 당사자 비노출) | (동일) | 완전 불가 |

> **데이터는 삭제하지 않는다.** 차단/탈퇴 시에도 서버의 데이터는 보존되며, 접근 및 UI 표시만 제어한다.

## 4. Baby 삭제 후 데이터 접근 정책

| 시점 | OWNER 접근 | 데이터 상태 |
|------|:--------:|----------|
| 삭제 직후 ~ 30일 이내 | 🔒 다운로드 전용 페이지 접근 가능 | 보존 중 |
| 삭제 후 31일 이상 | ❌ | 완전 삭제 |

## 5. 초대 상태별 접근 권한 요약

| 초대 상태 | 접근 가능 여부 |
|---------|--------------|
| 초대 발송됨 (토큰 유효, 미승인) | ❌ (승인 전은 접근 불가) |
| 초대 승인됨 (INVITED 상태) | ✅ (초대된 Baby 범위 내) |
| 초대 승인됨 (OWNER 역할로 초대) | ✅ (전체 관리 권한) |
| 초대 토큰 만료 (30분 초과) | ❌ (재초대 필요) |
| 초대 취소됨 | ❌ |
| 차단됨 | ❌ |
| 탈퇴됨 | ❌ |
