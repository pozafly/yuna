# Image Generation 스킬

**목적:** "Noah & June" 프로젝트의 레퍼런스를 바탕으로, 프론트엔드가 코딩을 시작하기 전 시각적 청사진(Mockup)을 제시하거나 프로젝트에 필요한 그래픽 에셋 아이디어를 제공합니다.

**사용 시점:**
- 새로운 UI 컴포넌트나 화면을 기획할 때 (예: "매우 둥근 모서리와 파스텔톤 배경의 편지 목록 뷰")
- 버튼, 스티커, 낙서 요소 등 에셋의 아이디어 스케치가 필요할 때

**사용 방법 (가이드):**
- Text-to-Image 프롬프트를 작성하여 이미지를 생성합니다.
- 프롬프트 작성 시 `docs/TECH-F_design-guidelines.md`에 명시된 핵심 디자인 요소(예: Chunky corners, Pastel colors 등)를 명확히 포함하세요.
