# Web Search 스킬

**목적:** 타 서비스의 정책 벤치마킹, 최신 개인정보보호법 등의 규정 확인, 알 수 없는 에러 해결책 확인 등을 위해 인터넷을 검색합니다.

**사용 시점:**
- Yuna 앱과 유사한 서비스(FamilyAlbum 등)의 기능/UI 벤치마킹이 필요할 때
- 최신 법규(COPPA 등)나 정책 트렌드를 파악해야 할 때
- 코드 작성 중 공식 문서나 커뮤니티(StackOverflow 등) 레퍼런스가 필요할 때

**사용 방법 (가이드):**
- 구체적인 키워드로 도구를 호출합니다.
- 검색된 내용 중 출처가 명확하고 신뢰성 있는 정보를 우선하여 반영합니다.
